Структура файла EIF (Ext2D Engine Image File)

  --------------------------------
  |       Секция       | Размер  |
  --------------------------------
  | Сигнатура, 'EIF_'  | 4 байта |
  --------------------------------
  | Ширина изображения | 4 байта |
  --------------------------------
  | Высота изображения | 4 байта |
  --------------------------------
  | Размер исходных    |         |
  | данных изображения | 4 байта |
  --------------------------------
  | Начало блока       |         |
  | данных, 'data'     | 4 байта |
  --------------------------------
  | Сжатые данные      |         |
  | изображения        |         |
  --------------------------------
