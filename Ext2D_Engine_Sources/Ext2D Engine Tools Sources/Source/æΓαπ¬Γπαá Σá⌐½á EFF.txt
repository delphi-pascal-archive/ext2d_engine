Структура файла EFF (Ext2D Engine Font File)

  --------------------------------
  |       Секция       | Размер  |
  --------------------------------
  | Сигнатура, 'EFF_'  | 4 байта |
  --------------------------------
  | Ширина изображения |         |
  | шрифта             | 4 байта |
  --------------------------------
  | Высота изображения |         |
  | шрифта             | 4 байта |
  --------------------------------
  | Размер исходных    |         |
  | данных изображения | 4 байта |
  --------------------------------
  | Размер шрифта      | 4 байта |
  --------------------------------
  | Данные о символах  | 1536    |
  | (256 символов)     | байт    |
  --------------------------------
  | Начало блока       |         |
  | данных, 'data'     | 4 байта |
  --------------------------------
  | Сжатые данные      |         |
  | изображения шрифта |         |
  --------------------------------
