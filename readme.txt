Простой двумерный движок для создания игр.
В составе есть функции для работы с графикой, звуком, устройствами ввода (клавиатура и мышь), физикой,
визуальными эффектами, а также функции для реализации выбора обьектов.
Движок работает только в полноэкранном режиме и глубине цвета 16 бит.
Для файлов изображений, звуков и шрифтов используется собственный формат *.eif, *.esf и *.eff соответственно.
Конвертеры прилагаются.
При разработке основной упор делался на простоту в освоении и использовании, а также отказ от VCL и
использование только WIN API + DirectX (для максимальной скорости работы).

Все ваши замечания, вопросы и предложения присылайте на zip000@yandex.ru. Прошу особенно не критиковать, т.к. это мой первый опыт создания движка. Движок пока еще не закончен и не отлажен до конца, поэтому скорее всего в нем есть ошибки :). Если вы их найдете, обязательно напишите мне.

Папки:
\Bin - там лежит бинарник;
\SDK - заголовочный файл, а также нескольно простых примеров использования движка с исходниками;
\Tools - утилиты, необходимые при использовании движка.

Для создания приложения с исползованием Ext2D Engine необходимо включить в список Uses заголовочный файл Ext2D и скопировать в папку с вашей программой сам DLL файл. Также вы можете использовать готовый шаблон приложения (находится в папке \SDK\Examples\Source\, файл E2DApp.dpr).

Замечания по движку.
1) Для всех загруженных изображений используется цветовой ключ (черный $0000);
2) Отсечение невидимых частей изображения при выводе его на экран производится автоматически;
3) Восстановление потеряных поверхностей также происходит автоматически.
